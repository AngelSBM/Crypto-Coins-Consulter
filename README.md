## Wanna run it?
Just have to download the code of the project and open the `index.html`

**The Crypto Coins available for consult are:**

- Bitcoin
- Etherium
- XRP
- Tether
- Chainlink
- Stellar
- Litecoin
- Porkadot
- Cardano
- Bitcoin Cash

**The Country Coins available for all the Crypto Coins are:**

- Dollar
- Dominican Peso
- Pound Sterling
- Mexican Peso
